Project Aon Base Package
8 June 2006
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This Base Package is the glue that helps you navigate your downloads
from Project Aon <http://www.projectaon.org/>. To start, open the file
named "index.htm" in your favorite web browser. This is usually as
simple as double clicking on the file icon.

From there, you will be presented with a list of books currently
available from Project Aon. If you click on one of the links and your
browser reports that the file is not found, please visit the Project
Aon website and download the proper file.
