[1] (Weapons) It is reasonably clear that the Machine Pistol should not be used if you have fewer than 6 rounds of 9mm ammunition.

For some readers, however, it may seem counterintuitive to have ammunition left but be unable to use the weapon. We suggest to these readers that when 9mm ammunition falls below 6 rounds, the Machine Pistol can be treated as a Pistol (cf. Section 333). While this is not a perfect solution, it is a reasonable compromise.
